Components

Components are the building blocks of Angular applications. Each component consists of:

  • A TypeScript class
  • An HTML template
  • Optional CSS styles

Example

ng generate component hello
  

import { Component } from '@angular/core';

@Component({
  selector: 'app-hello',
  template: '<h2>Hello, Angular!</h2>'
})
export class HelloComponent {}

Use <app-hello></app-hello> in your template to display the component.
